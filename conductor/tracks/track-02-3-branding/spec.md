# Track 02.3: Branding & Logo

**Goal:** Design and implement the app icon and logo.

## Scope
1.  **Design:** Create a minimalist Vector Drawable logo.
    -   Concept: Abstract Loop / Pulse.
    -   Primary Color: `#4CAF50` (Green).
2.  **Implementation:**
    -   Update `ic_launcher_foreground.xml`.
    -   Update `ic_launcher_background.xml`.
    -   Add `Logo` component to the Home Screen header.

## Success Criteria
-   App icon looks professional on the device/emulator.
-   Home Screen displays the custom logo instead of just text.
